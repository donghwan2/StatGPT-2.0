page79. 관심변수와 설명변수
# 관심 변수(Interest Variable) : 예측하고자 하는 대상
= 종속변수, 타깃변수, 출력변수
(e.g.) 보험료, 성적, 아들키

# 설명 변수(Explanatory Variable) : 관심변수에 영향을 미치는 요인. 
= 독립변수, 피쳐(feature), 입력변수, 조절변수
(e.g.) 나이, 성별, 흡연여부, 공부시간, 아빠키


page82. 선형회귀 모델의 이해
# 단순 선형회귀(simple linear regression) : 수치형 관심변수를 수치형 설명변수의 정비례로 설명하는 모델
𝑌=𝑎+𝑏𝑋

# 단순 선형회귀 적합 예제 : 아빠키와 아들키의 관계 𝑦=86.07+0.514𝑥
=> 기울기와 절편이 정해지면 아들키와 아빠키 관계를 설명하는 선형회귀 모델을 만들 수 있습니다.

# 일반적인 선형회귀 모델 : 𝒚=𝛽_0+𝛽_1 𝒙_𝟏+𝛽_2 𝒙_𝟐+⋯+𝛽_𝑝 𝒙_𝒑
𝛽_0 : 절편(상수항),
𝛽_𝑖  : 회귀계수
설명변수(독립변수)의 회귀계수에 대한 t-검정 결과 p-value<0.05 일 경우 해당 설명변수는 종속변수 y와 연관성이 있다고 판단된다.

page84. 결정계수(R2)
# 결정계수(R2, R-square) : 모델의 데이터 설명력
모델이 데이터를 얼마나 잘 설명하는가에 대한 지표(설명력). 클수록 좋은 모델(0.8 이상이면 어느정도 큰 모델).
단순회귀의 경우 "상관계수의 제곱 = 결정계수".
관심변수 𝒚 분산과 모델 설명 분산을 비율로 계산.
SST=SSR/SST

page86. 선형회귀 통계 해석
# R-squared(설명력) : 모델의 데이터 설명력(에러를 모델이 얼마나 개선했는가)
- 클수록 좋은 모델(0.8 이상이면 충분히 큰 모델)
- Adj. R-suared가 더욱 적절한 평가 기준이다. 그 이유는 독립 변수가 많아지면 R-squared는 무조건 증가하는데, 변수의 개수까지 고려한 것이 Adj. R-suared이기 때문.

# Coefficient(회귀계수) : 회귀선의 기울기
- 독립변수가 한 단위 변화함에 따라 종속변수에 미치는 영향력 크기
- 특정 독립 변수의 회귀계수가 크면, 일반적으로 독립 변수 값이 조금만 커져도 종속 변수가 크게 변하게 된다. 

# P-value, 유의확률 : 신뢰할 수 있는 결과인가? 에 대한 척도
- 일반적으로 유의수준 0.05 미만이면  해당 변수가 종속변수와 유의한 관계가 있다고 판단 (0.05 이상이면 유의하지 않음)
